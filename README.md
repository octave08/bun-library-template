## Getting started
```bash
$ bun install
```
```bash
$ bun test
```

### Run package on example

#### root project
Go to `/` and do follow the steps below
```bash
$ bun link
```
```bash
$ bun watch
```

#### example project
Go to `/example` and do follow the steps below
```bash
$ bun link bun-library-template
```
```bash
$ bun dev
```
